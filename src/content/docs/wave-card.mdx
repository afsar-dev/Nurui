### Props

| Prop          | Type     | Default    | Description                              |
| ------------- | -------- | ---------- | ---------------------------------------- |
| `tags`        | `string` | `Required` | Tag or label displayed above the title.  |
| `title`       | `string` | `Required` | Main title text displayed on the card.   |
| `description` | `string` | `Required` | Description text shown below the title.  |
| `buttonText`  | `string` | `Required` | Text displayed inside the action button. |
