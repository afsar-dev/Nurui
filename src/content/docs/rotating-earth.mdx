`RotatingEarth`

| Prop        | Type     | Default | Description                                                                |

| `height`    | `number` | `600`   | Height of the Earth canvas container. Responsive limits are applied.       |
| `className` | `string` | `""`    | Additional Tailwind/utility classes for customizing the wrapper container. |
