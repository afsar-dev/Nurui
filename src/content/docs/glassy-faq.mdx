### Props

| Prop             | Type         | Default | Description                                                              |
| ---------------- | ------------ | ------- | ------------------------------------------------------------------------ |
| `question`       | `string`     | —       | The FAQ question text.                                                   |
| `answer`         | `string`     | —       | The corresponding answer text for the question.                          |
| `questionNumber` | `number`     | —       | The index/number of the question used for ordering and identification.   |
| `openKey`        | `number`     | —       | The currently opened question number. Used to determine expand/collapse. |
| `setOpenKey`     | `() => void` | —       | Function to update the `openKey` state when toggling a question.         |
