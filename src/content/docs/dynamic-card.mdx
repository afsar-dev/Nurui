### Props

| Prop            | Type     | Default    | Description                                               |
| --------------- | -------- | ---------- | --------------------------------------------------------- |
| `normalTitle`   | `string` | `Required` | Regular part of the title, usually styled in plain text.  |
| `colorfulTitle` | `string` | `Required` | Highlighted part of the title, usually styled with color. |
| `description`   | `string` | `Required` | Description text shown below the title.                   |
| `buttonText`    | `string` | `Required` | Text displayed inside the action button.                  |
