### Props

`MobileMenuContext`

| Prop          | Type                                            | Default    | Description                              |
| ------------- | ----------------------------------------------- | ---------- | ---------------------------------------- |
| `showMenu`    | `boolean`                                       | `true`     | Indicates if the mobile menu is visible. |
| `setShowMenu` | `React.Dispatch<React.SetStateAction<boolean>>` | `() => {}` | Function to update `showMenu`.           |

`FutureButton Props`

| Prop                 | Type                              | Default     | Description                                                                    |
| -------------------- | --------------------------------- | ----------- | ------------------------------------------------------------------------------ |
| `shape`              | `"default" \| "flat" \| "simple"` | `"default"` | Determines the button style and path rendering.                                |
| `className`          | `string`                          | `undefined` | Additional class names for the button.                                         |
| `children`           | `React.ReactNode`                 | `undefined` | Button content.                                                                |
| `enableBackdropBlur` | `boolean`                         | `false`     | Enables backdrop blur effect for the button frame.                             |
| `enableViewBox`      | `boolean`                         | `false`     | Enables the SVG viewBox for responsive scaling.                                |
| `customPaths`        | `string[]`                        | `undefined` | Custom SVG path definitions (JSON strings) instead of default paths.           |
| `bgColor`            | `string`                          | `undefined` | Optional background color for the button (not used in default implementation). |
| `textColor`          | `string`                          | `undefined` | Text color for the button; defaults to theme color.                            |
| `...props`           | `React.ComponentProps<"button">`  | `undefined` | Standard button HTML attributes like `onClick`, `type`, `disabled`, etc.       |

`Frame Props`

| Prop                 | Type                                | Default     | Description                                                                            |
| -------------------- | ----------------------------------- | ----------- | -------------------------------------------------------------------------------------- |
| `paths`              | `Paths` (array of path definitions) | required    | Array of objects describing SVG paths, including `show`, `style`, and `path` commands. |
| `enableBackdropBlur` | `boolean`                           | `false`     | Adds a blur filter behind paths when true.                                             |
| `className`          | `string`                            | `undefined` | Additional class names for the SVG wrapper.                                            |
| `...props`           | `React.ComponentProps<"svg">`       | `undefined` | Standard SVG element props, e.g., `width`, `height`, `viewBox`.                        |
