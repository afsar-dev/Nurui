### Props

| Prop Name      | Type                           | Default        | Description                                                               |
| -------------- | ------------------------------ | -------------- | ------------------------------------------------------------------------- |
| `children`     | `React.ReactNode`              | `undefined`    | Content to be rendered inside the drawing area.                           |
| `strokeColor`  | `string`                       | `"#FF9900"`    | Color of the drawn lines.                                                 |
| `strokeWidth`  | `number`                       | `10`           | Thickness of the stroke/line in pixels.                                   |
| `type`         | `"drawOnHold" \| "drawAlways"` | `"drawAlways"` | Drawing mode: only on mouse hold (`drawOnHold`) or always (`drawAlways`). |
| `followEffect` | `boolean`                      | `false`        | If `true`, each drawn stroke fades out smoothly (like a trailing effect). |
