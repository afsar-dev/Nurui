### Props

| Prop        | Type       | Default                                                                                               | Description                                                        |
| ----------- | ---------- | ----------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------ |
| `colors`    | `string[]` | `["#00FF87", "#0CFD95", "#17FBA2", "#23F9B2", "#30F7C3", "#3DF5D4", "#45F4DE", "#53F1F0", "#60EFFF"]` | Array of colors for the ripple cells.                              |
| `cellSize`  | `number`   | `52`                                                                                                  | Size in pixels of each ripple cell.                                |
| `cells`     | `number`   | `3`                                                                                                   | Number of cells per row/column (creates a `cells x cells` grid).   |
| `delayStep` | `number`   | `100`                                                                                                 | Delay in milliseconds between the animations of consecutive cells. |
