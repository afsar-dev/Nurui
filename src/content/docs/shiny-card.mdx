### Props

| Prop         | Type                  | Default | Description                                            |
| ------------ | --------------------- | ------- | ------------------------------------------------------ |
| featureName  | `string`              | `–`     | Title of the feature section displayed on the card.    |
| featureItems | `string[]`            | `–`     | List of feature descriptions to be rendered in a list. |
| icon         | `ReactNode \| string` | `–`     | An icon or string displayed at the top of the card.    |
