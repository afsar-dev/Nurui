### Props

| Prop           | Type     | Default | Description                                           |
| -------------- | -------- | ------- | ----------------------------------------------------- |
| `className`    | `string` | —       | Optional custom class name for the container div.     |
| `onBlurTitle`  | `string` | —       | Placeholder text shown when the input is not focused. |
| `onFocusTitle` | `string` | —       | Placeholder text shown when the input is focused.     |
| `buttonTitle`  | `string` | —       | The text displayed inside the submit button.          |
