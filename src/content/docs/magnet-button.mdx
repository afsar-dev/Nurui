### Props

| Prop            | Type                                            | Default | Description                                                                   |
| --------------- | ----------------------------------------------- | ------- | ----------------------------------------------------------------------------- |
| `particleCount` | `number`                                        | `12`    | Number of particles that animate around the button.                           |
| `attractRadius` | `number` _(optional, not implemented yet)_      | `-`     | Intended to control attraction radius. Currently not used in logic.           |
| `className`     | `string`                                        | `-`     | Additional Tailwind or CSS classes for custom styling.                        |
| `children`      | `React.ReactNode`                               | `-`     | Content passed inside the button (inherited from native props).               |
| `...props`      | `React.ButtonHTMLAttributes<HTMLButtonElement>` | `-`     | All other native `<button>` props such as `onClick`, `type`, `disabled`, etc. |
