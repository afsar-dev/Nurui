### Props

| Prop          | Type     | Default                  | Description                         |
|---------------|----------|--------------------------|-------------------------------------|
| `text`        | string   | `"your@email.com"`       | The text/email copied to clipboard  |
| `defaultLabel`| string   | `"Copy my email address"`| Label before copy                   |
| `successLabel`| string   | `"Email is Copied!"`     | Label after copy                    |
